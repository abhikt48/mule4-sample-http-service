# Overview
Sample Mule4 application with HTTP endpoint. You can learn to easily expose a service with HTTP endpoint and how can handle user input e.g. query parameter input.

## Prerequiste Setup
1. Java 8 
2. Maven
3. Anypoint Studio 7.6.0
